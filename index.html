Besiege

Besiege is a real-time, single-player strategy game I developed for Android and desktop. It uses a procedurally generated map and a modified visibility graph for efficient, guaranteed shortest-path pathfinding. It was written in Java, and is still in alpha development. The source code and an updated timeline of my progress can be found here. If you're feeling adventurous, play the demo version here.

  • Date: June 2013 - present

FindA

FindA is an iPhone application I developed with three of my classmates to instantly locate any student's dorm room, professor's office, classroom, printer, or bathroom on campus. I specialized in writing Python scripts to organize and clean data scraped from six different websites and compile it into a SQLite database, as well as managing our Apache server. You can see a detailed timeline of our progress here, or check out our main site here.

  • Date: January - May 2014
  • Team: Curtis Belmonte '16, Emily Hsu '16, Lucas Mayer '16

  • Class: Advanced Programming Techniques with Professor Brian Kernighan

World War Q

World War Q is a 3D flight combat simulator I developed with my roommate. The game features realistic physics, an intelligent AI, and a variety of aircraft and environments. It is written in C++ and was almost entirely pair-programmed. Click here to see gameplay footage, or view the source code here.

  • Date: May 2014
  • Partner: Jason Altschuler '16

Dank or Jank

Dank or Jank is an online music player and recommendation application I developed with three classmates for HackPrinceton2013. It uses SoundCloud's API to play and rank over 50,000 new songs. It was written mostly in JavaScript. Check out the source code here, or try it for yourself here

  • Date: November 2013
  • Team: Curtis Belmonte '16, Jeremy Cohen '16, Lucas Mayer '16

FalconVR

FalconView is a geospatial mapping program developed by Georgia Tech Research Institute. While working at GTRI's Information and Communications Lab, I integrated virtual reality capabilities into FalconView with the Oculus Rift, creating a highly immersive way to explore 3D maps.

  • Date: Summer 2014
  • Organization: GTRI ICL

Internet Voting

My Independent Research at Princeton focused on Internet Voting and cryptographic voting protocols. I presented a literary review entitled "Challenges for Large-Scale Internet Voting Implementations". I also made a short video that distilled complex Internet security concepts into everyday terms, called "Should we trust Internet Voting?"

  • Date: January 2015-Present
